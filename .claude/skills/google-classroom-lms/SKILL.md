# Google Classroom & Gemini Integration

## Google APIs Used

| API | Purpose |
|-----|---------|
| Classroom API | Create assignments, topics, sync roster, per-student assignment |
| Drive API | Push lesson plans to admin folder, file management |
| Calendar API | Sync lesson plan events to teacher's Google Calendar |
| Slides API | Create presentations (content generated by Gemini) |
| Forms API | Create auto-grading quizzes (content by Gemini) |

## Pushing Interactive Activities to Classroom

```python
def create_interactive_assignment(credentials, course_id, activity):
    service = build("classroom", "v1", credentials=credentials)
    coursework = {
        "title": activity.title,
        "description": f"Complete this interactive activity: {activity.description}",
        "materials": [{"link": {"url": activity.cloudfront_url}}],
        "maxPoints": activity.point_value,
        "workType": "ASSIGNMENT",
        "state": "PUBLISHED"
    }
    # For accommodation versions: assign modified version to specific students
    if activity.accommodation_student_ids:
        coursework["assigneeMode"] = "INDIVIDUAL_STUDENTS"
        coursework["individualStudentsOptions"] = {
            "studentIds": activity.accommodation_student_ids
        }
    return service.courses().courseWork().create(
        courseId=course_id, body=coursework).execute()
```

## Gemini Imagen for Video Illustrations

```python
import google.generativeai as genai

def generate_scene_illustration(scene_description, style="educational"):
    model = genai.ImageGenerationModel("imagen-3.0-generate-002")
    response = model.generate_images(
        prompt=f"{scene_description}, {style} illustration style, clean background",
        number_of_images=1,
        aspect_ratio="16:9"  # video widescreen
    )
    return response.images[0]  # PIL Image → save as PNG for ffmpeg
```

## Calendar Sync (All Optional)

```python
def sync_to_google_calendar(credentials, plan, calendar_id="primary"):
    service = build("calendar", "v3", credentials=credentials)
    for day in plan.daily_plans:
        event = {
            "summary": f"{day['title']} ({day['output_template_id']})",
            "description": f"Standards: {', '.join(day['standards_targeted'])}",
            "start": {"date": day["date"]},
            "end": {"date": day["date"]}
        }
        service.events().insert(calendarId=calendar_id, body=event).execute()
```

## Key Rules

1. OAuth tokens encrypted AES-256 — never in plaintext
2. STUDENT_COPY share mode for Slides assignments
3. Gemini for Slides/Forms content generation — Claude for everything else
4. Imagen for video illustrations — one image per scene, 16:9 aspect ratio
5. Per-student assignment in Classroom for accommodation versions
6. Calendar sync is optional — teacher enables in Settings
7. Admin Drive folder auto-organized: Teacher Name > Week N
